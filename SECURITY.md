# MovieSnapEditor セキュリティ説明

## 1. このアプリのデータの流れ

MovieSnapEditor は**すべてブラウザ内で処理が完結**します。

```
ユーザーが動画を選択
      ↓
ブラウザ（Safari / Chrome）内でフレーム抽出
      ↓
ZIPファイルをローカルに保存
```

動画ファイルは外部サーバーに送信されません。処理はすべてデバイス内で行われます。

---

## 2. アクセスできる情報・できない情報

| 情報 | アクセス可否 | 理由 |
|---|---|---|
| ユーザーが選んだ動画ファイル | ○ 可能 | ユーザーが明示的に選択した場合のみ |
| 他の写真・動画 | × 不可能 | iOSサンドボックスで制限 |
| 連絡先・メール・他アプリのデータ | × 不可能 | iOSサンドボックスで制限 |
| ファイルシステムへの直接アクセス | × 不可能 | iOSサンドボックスで制限 |

**ユーザーが意図的に渡したものだけ**がアプリに届きます。それ以外のiPhone内の情報は構造的に取得できません。

---

## 3. ブラウザのセキュリティ機構

### サンドボックス
ブラウザはサンドボックス（隔離された実行環境）の中で動作します。悪意あるサイトにアクセスしても、ブラウザの外（他のアプリやOSのデータ）への影響を封じ込めます。アクセスを「防ぐ」のではなく、「被害を閉じ込める」仕組みです。

### 同一オリジンポリシー（Same-Origin Policy）
異なるドメインのサイト同士はお互いのデータに一切アクセスできません。MovieSnapEditorのページ内のデータを、他のサイトが読み取ることは不可能です。

### HTTPS
`https://asmintt.github.io/MovieSnapEditor/` の `https` により、通信が暗号化されています。通信経路での盗み見はできません。

---

## 4. iPhoneでの安全性

iOSの追加保護：

- Safariの各タブはプロセスが分離されている
- ブラウザアプリ自体がiOSのサンドボックス内で動作する
- デスクトップ版と異なり、悪意ある拡張機能が入り込む余地がほぼない

### 自分のiPhone vs 知らないPC

| | 自分のiPhone | 知らないPC |
|---|---|---|
| マルウェア感染リスク | 低い | 不明 |
| キーロガー（入力盗聴） | ほぼなし | 混入している可能性あり |
| ブラウザ拡張機能 | ほぼなし | 怪しいものが入っている可能性あり |
| OSの管理者 | 自分 | 不明 |

自分が管理しているiPhoneでの操作が最も安全です。

---

## 5. GitHub Pages（公開サイト）のセキュリティ

### GitHub 2段階認証（2FA）
GitHubアカウントに2FAを設定しています。コードを書き換えるにはパスワード＋認証アプリの両方が必要なため、パスワードが漏れただけでは改ざんできません。

```
パスワードのみ → ログイン不可（2FAが防ぐ）
パスワード ＋ 認証アプリのコード → ログイン可能
```

### Publicリポジトリのリスク
- ソースコードは誰でも閲覧可能
- URLを知っていれば誰でもアプリを使用可能
- ただしアプリがサーバーにデータを送信しないため、他者の利用による情報流出はない

## 6. 外部ライブラリ（CDN）について

`index.html` で以下の外部ライブラリをCDN経由で読み込んでいます：

```html
<script src="https://cdnjs.cloudflare.com/.../jszip.min.js"></script>
<script src="https://cdnjs.cloudflare.com/.../FileSaver.min.js"></script>
```

**用途：**
- JSZip → 抽出した画像をZIPファイルにまとめる
- FileSaver.js → ZIPファイルのダウンロードダイアログを出す

**CDNリスク：** Cloudflareのサーバーが侵害された場合、悪意あるコードが混入する理論上のリスクがあります。ただしCloudflareは世界最大級のインフラ会社であり、実用上は無視できるレベルです。

**動画処理部分（フレーム抽出・Canvas API）は外部ライブラリに依存しておらず、CDNリスクはゼロです。**

---

## 7. Windowsでの利用

Windows Defenderをはじめとするウイルス対策ソフトとの競合はありません。

| 要素 | 判定 | 理由 |
|---|---|---|
| `github.io` のURL | 問題なし | 信頼済みドメイン |
| ブラウザ内JavaScript | 対象外 | サンドボックス内で完結 |
| ダウンロードされるZIP | 問題なし | 中身はJPEG画像のみ |
| 実行ファイル（.exe等） | 存在しない | インストール不要なWebアプリ |

---

## 8. セキュリティ対策まとめ

| 脅威 | 対策 |
|---|---|
| コードの改ざん | GitHub 2FA |
| 通信の盗み見 | HTTPS |
| 他サイトからのデータ盗取 | 同一オリジンポリシー |
| 端末への不正アクセス | iOSサンドボックス |
| 不正ログイン | GitHubパスワード ＋ 2FA |

個人ツールとして使用する範囲において、セキュリティは十分に確保されています。
